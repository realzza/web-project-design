CSS Transitions & Animations

The transition properties in CSS allow you to change property values smoothly over a given duration. They are generally used in combination with psuedo-classes like :hover and :active or :focus, which specify rules based on user interaction with page elements.

Transitions and animations can be a good way to provide feedback to a user about clickable content or focus attention.

Kirupa.com provides one of the most exhaustive intros to CSS animation. Check out the stylesheet referenced in this document for css code and more links.

There are also animations properties, which are a bit more complicated. Just remember they take two steps. First, you need to specify an @keyframes ____ {} rule with a name and the property values to animate in their starting states or ending states. And then that name is used to bind an animation to an element or psuedo-class.

Usually, the animation will only play once, unless you specify an animation-iteration-count:. You can set this to a numerical value or "infinite".
